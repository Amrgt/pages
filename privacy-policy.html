Privacy Policy

YouTube Summarizer — Chrome Extension · Last updated: February 27, 2026

TL;DR: We don't collect, store, or sell your personal data. Video transcripts are sent directly from your browser to Google's Gemini API using your own API key — we never see them.

1. Introduction

This Privacy Policy describes how YouTube Summarizer ("the Extension") handles information when you use our Chrome extension. We are committed to protecting your privacy and being transparent about our practices.

2. Data Stored Locally

The following data is stored only on your device using Chrome's built-in storage APIs — none of it is transmitted to us:

  • API key — Your Google Gemini API key, stored in chrome.storage.local. It leaves your browser only when sent directly to Google's API.
  • Language preferences — Your preferred summary languages, stored in chrome.storage.sync.
  • Cached results — Previously generated summaries and fact-checks, stored locally to avoid redundant API calls.
  • Consent flag — Whether you accepted the data-sharing notice.
  • No personal data: We do not collect your name, email, browsing history, or any personally identifiable information.
  • No tracking: No analytics, no cookies, no telemetry.

3. Data Sent Externally

When you click "Summarize this" or "Fact Check" on a YouTube video, the video's transcript is sent directly from your browser to Google's Gemini API (generativelanguage.googleapis.com) for processing, using your own API key.

The Extension does not route data through any intermediary server. The only external communication is between your browser and Google's API.

4. Third-Party Services

The Extension interacts only with:

  • YouTube (youtube.com) — To read video transcripts directly from the page.
  • Google Gemini API — To generate summaries and fact-checks. Governed by Google's API Terms of Service.

5. Data Retention

All data is stored locally on your device and can be removed at any time by:

  • Removing the Extension from Chrome
  • Clearing extension data via chrome://extensions

6. Permissions

The Extension requests only the minimum Chrome permissions necessary to function:

  • storage — To save your API key, language preferences, and cached results locally.
  • Host permission for generativelanguage.googleapis.com — To call Google's Gemini API.
  • Content script on youtube.com — To inject the Summarize button and read video transcripts from the page.

7. Children's Privacy

The Extension is not directed at children under 13. We do not knowingly collect information from children.

8. Changes to This Policy

We may update this Privacy Policy from time to time. Any changes will be reflected on this page with an updated "Last updated" date. The Extension's next release will also reflect any changes.
